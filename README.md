# Project Skeleton
Minimalistic project structure for ML projects in PyTorch on AWS.

## Usage

```
git clone https://github.com/josiahdavis/project-skeleton.git
cd project-skeleton
pip install -e .
```

## Features
- [x] cookiecutter-like directory structure
- [x] pip install-able
- [ ] s3 utilities
- [ ] pytest
- [ ] sphinx
